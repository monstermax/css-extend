CSS Extend   -   extend css classes with other classes

CSS Extend allow to extend css classes with other classes. 

    <style>
    h3 {
      color: green;
      content: "@extend .upper-case .text-arial";
    }

    h4 {
      content: "@extend .bg-dark .text-red .text-arial";
    }

    h5 {
      content: "@extend h4";
      color: yellow;
    }


    .text-arial {
      font-family: arial;
    }

    .upper-case {
      text-transform: uppercase;
    }

    .text-red {
      color: red;
    }

    .bg-dark {
      background-color: darkgrey;
    }
    </style>

h3 (arial & uppercase)

h4 (dark background & red text & arial)

h5 (all 'h4' class properties but yellow color, and the original h5 fontsize)
