CSS Extend   -   extend css classes with other classes

CSS Extend allow to extend css classes with other classes. 

    <style>
    h3 {
      color: green;
      --extends: .upper-case .text-arial;
        /* the 'h3' class will inherit the .upper-case class and the .text-arial class */
    }

    h4 {
      --extends: .bg-dark .text-red .text-arial;
    }

    h5 {
      --extends: h4;
      color: yellow;
    }


    .text-arial {
      font-family: arial;
    }

    .upper-case {
      text-transform: uppercase;
    }

    .text-red {
      color: red;
    }

    .bg-dark {
      background-color: darkgrey;
    }
    </style>

h3 (arial & uppercase)

h4 (dark background & red text & arial)

h5 (all 'h4' class properties but yellow color, and the original h5 fontsize)

IS TRANSFORMED TO

    <style>
    h3 {
      color: green;
      text-transform: uppercase;
      font-family: arial;
    }

    h4 {
      background-color: darkgrey;
      color: red;
      font-family: arial;
    }

    h5 {
      color: yellow;
      background-color: darkgrey;
      font-family: arial;
    }


    .text-arial {
      font-family: arial;
    }

    .upper-case {
      text-transform: uppercase;
    }

    .text-red {
      color: red;
    }

    .bg-dark {
      background-color: darkgrey;
    }
    </style>
